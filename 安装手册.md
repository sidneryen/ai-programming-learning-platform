# AI 智能编程学习系统 - Ubuntu Server 22.04 安装部署手册

## 一、环境准备

### 1.1 更新系统
```bash
sudo apt update && sudo apt upgrade -y
```

### 1.2 安装基础依赖
```bash
sudo apt install -y git python3 python3-pip python3-venv nginx certbot python3-certbot-nginx
```

### 1.3 安装编程语言编译环境（支持多语言代码评测）
系统支持 Python、C、C++、Java、Go 等语言的代码编译和运行，请安装以下环境：

```bash
# 安装 C/C++ 编译环境
sudo apt install -y gcc g++ build-essential

# 安装 Java JDK（用于 Java 代码编译和运行）
sudo apt install -y openjdk-17-jdk
# 设置 JAVA_HOME 环境变量（将以下内容添加到 /etc/environment 或 ~/.bashrc）
# export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
# export PATH=$JAVA_HOME/bin:$PATH

# 安装 Go 语言环境
sudo apt install -y golang-go
# 验证安装
go version
```

### 1.4 安装 Node.js（前端构建）
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

## 二、项目部署

### 2.1 创建项目目录
```bash
sudo mkdir -p /opt/ai-programming-platform
sudo chown $USER:$USER /opt/ai-programming-platform
cd /opt/ai-programming-platform
```

### 2.2 克隆项目代码
```bash
git clone https://github.com/sidneryen/ai-programming-learning-platform.git .
```

---

## 三、后端配置

### 3.1 创建虚拟环境
```bash
cd /opt/ai-programming-platform
python3 -m venv venv
source venv/bin/activate
```

### 3.2 安装 Python 依赖
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 3.3 配置环境变量
```bash
cp .env.example .env
nano .env
```

编辑 `.env` 文件，填入您的 DeepSeek API Key：
```ini
DEEPSEEK_API_KEY=your_deepseek_api_key_here
SECRET_KEY=your-secret-key-change-in-production
FLASK_APP=app.py
FLASK_ENV=production
```

### 3.4 测试后端服务
```bash
python app.py
```
验证服务是否正常启动（默认端口 5002），按 `Ctrl+C` 停止。

---

## 四、前端构建

### 4.1 进入前端目录
```bash
cd /opt/ai-programming-platform/frontend
```

### 4.2 安装 Node.js 依赖
```bash
npm install
```

### 4.3 修改 API 配置
```bash
nano src/services/api.js
```

将 `baseURL` 改为后端服务地址：
```javascript
baseURL: 'http://localhost:5002/api'
```

### 4.4 构建生产版本
```bash
npm run build
```

### 4.5 复制构建产物到 Nginx 目录
```bash
sudo cp -r dist/* /var/www/html/
```

---

## 五、Nginx 配置

### 5.1 创建 Nginx 配置文件
```bash
sudo nano /etc/nginx/sites-available/ai-programming-platform
```

配置内容：
```nginx
server {
    listen 80;
    server_name your-domain.com;  # 替换为您的域名

    # 前端静态文件
    location / {
        root /var/www/html;
        try_files $uri $uri/ /index.html;
    }

    # API 反向代理
    location /api/ {
        proxy_pass http://localhost:5002/api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### 5.2 启用配置并测试
```bash
sudo ln -s /etc/nginx/sites-available/ai-programming-platform /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

```

```

## 七、配置后端服务自启动

### 7.1 创建 Systemd 服务文件
```bash
sudo nano /etc/systemd/system/ai-programming-backend.service
```

服务配置：
```ini
[Unit]
Description=AI Programming Learning Platform Backend
After=network.target

[Service]
User=www-data
WorkingDirectory=/opt/ai-programming-platform
Environment="PATH=/opt/ai-programming-platform/venv/bin:/usr/bin:/usr/local/bin:/usr/lib/jvm/java-17-openjdk-amd64/bin:/usr/lib/go/bin"
Environment="JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64"
Environment="GOROOT=/usr/lib/go"
Environment="GOPATH=/opt/go"
ExecStart=/opt/ai-programming-platform/venv/bin/python app.py
Restart=always

[Install]
WantedBy=multi-user.target
```

### 7.2 启用并启动服务
```bash
sudo systemctl daemon-reload
sudo systemctl enable ai-programming-backend
sudo systemctl start ai-programming-backend
```

### 7.3 检查服务状态
```bash
sudo systemctl status ai-programming-backend
```

## 注意事项

1. **DeepSeek API Key**：必须从 DeepSeek 官网获取有效密钥
2. **域名**：部署前需准备域名并解析到服务器 IP
3. **数据库**：项目使用 SQLite，数据文件为 `learning_analysis.db`
4. **备份**：定期备份数据库文件和配置文件
5. **默认账号**：
   - **教师账号**：用户名 `teacher`，密码 `teacher123`
   - **学生账号**：用户名 `学号`，初始密码 `123456`（由 Excel 导入时自动设置）
6. **编程语言环境**：系统支持 Python、C、C++、Java、Go 等语言的代码评测，部署时需确保已安装相应编译环境（详见 1.3 节）

## 十一、初始化教师账号（如需重置）

首次部署时，系统会自动创建默认教师账号。如需手动重置，可执行以下命令：

```bash
cd /opt/ai-programming-platform
source venv/bin/activate
python -c "
from app import app, init_default_teacher
with app.app_context():
    init_default_teacher()
"
```

## 十二、重置所有学生密码

如果需要将所有学生密码重置为 `123456`，可调用以下 API 接口：

```bash
curl -X POST http://localhost:5002/api/students/reset-passwords
```

或通过教师端管理界面操作。
